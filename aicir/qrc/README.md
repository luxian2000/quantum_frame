# QRC 模块说明

`aicir.qrc` 是量子储备池计算（Quantum Reservoir Computing, QRC）模块。

> **当前状态：占位骨架。** 仅有包入口，尚无公开 API。本文件记录该子系统的
> 定位与计划结构，随实现逐步填充。

## 定位

QRC 用一个**固定（不训练）的量子动力学系统**作为“储备池”：把输入信号注入
量子系统，让其在高维希尔伯特空间中自然演化产生丰富的非线性特征，再用一个
**可训练的经典读出层**（如线性回归/岭回归）从测量结果映射到目标输出。

与本仓库已有的 VQA 路线（VQE/QAOA/QAS）不同：QRC **不优化线路参数**，只训练
经典读出，因此训练成本低，天然适合时间序列预测、时序分类等时序任务。

## 计划结构

随实现推进，预期分层（暂定，可调整）：

- 储备池（reservoir）：基于 `aicir.core` 的线路/态演化构造固定动力学；输入注入与
  时间步推进。
- 读出（readout）：经典可训练读出层（线性/岭回归等），消费储备池的测量特征。
- 任务接口：时序数据集封装与 `fit`/`predict` 风格入口。

## 约定

- 线路构建与仿真复用 `aicir` 后端抽象（`StateVector`/`DensityMatrix`、门构造器），
  不引入外部量子 SDK。
- 注释/文档为中文，遵循仓库风格。
